Methodology & Disclaimer

Benchmarks performed using BenchmarkDotNet precision profiling on Intel i7-13700KF with .NET 8.0. Results represent comparative performance between Omni Collections and standard .NET baseline implementations under controlled test conditions. Actual performance may vary based on hardware, software environment, and specific use cases.

All benchmark data is reproducible and available in the repository. Performance improvements are measured as baseline execution time divided by Omni Collections execution time.
